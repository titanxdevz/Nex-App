## Full Description (4000 chars)
NexChat is a modern, secure messaging app built for seamless communication.

**KEY FEATURES:**
- Direct messaging with friends and family
- Group chats with admin controls and permissions
- Communities with events, roles, leaderboards & moderation
- Voice & video calls powered by LiveKit
- Stories (status updates) with photos, videos & text
- Message reactions, polls, and pinned messages
- Disappearing messages with configurable timers
- Message scheduling — send later
- Forward messages between conversations
- Starred messages for quick access
- Typing indicators and read receipts
- Online presence status
- File, image, video & voice message sharing
- Link previews with rich cards
- @mentions in group chats
- Global search across all conversations
- Custom notification preferences per conversation
- Message translation
- App lock with PIN and biometric authentication
- End-to-end encrypted local storage

**PRIVACY & SECURITY:**
- Encrypted token storage (AES-256-GCM)
- Certificate pinning
- Root/tamper detection
- Screenshot blocking on sensitive screens
- Biometric app lock
- No data backup to cloud
- Auto-clear clipboard

**BUILT WITH:**
- Kotlin + Jetpack Compose
- Material Design 3 with glassmorphism UI
- Room database for offline caching
- Socket.IO for real-time messaging
- Firebase Cloud Messaging for push notifications
- LiveKit for voice/video calls

Download NexChat today and experience messaging the way it should be.
